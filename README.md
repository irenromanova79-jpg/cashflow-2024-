# Отчёт о движении денежных средств · 2024

Интерактивный дашборд и формализованный отчёт о ДДС, построенный на основе данных учётной системы.

## Возможности

- **Фильтр по периоду**: год / месяц / неделя с выбором конкретного периода
- **KPI-карточки**: поступления, выбытия, чистый поток, дивиденды
- **Графики**: столбчатый по периодам, структура расходов (doughnut), чистый поток
- **Детализация**: сводная таблица, расходы по статьям с долями, контрагенты по неделям
- **Формальный отчёт ДДС**: три раздела (операционная, инвестиционная, финансовая деятельность) по месяцам
- Поддержка светлой и тёмной темы

## Запуск

### Локально
Просто откройте `index.html` в браузере — файл полностью автономный, без зависимостей кроме Chart.js (загружается из CDN).

### GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите `index.html` в корень репозитория
3. Перейдите в **Settings → Pages**
4. В разделе **Source** выберите ветку `main`, папку `/ (root)`
5. Нажмите **Save**
6. Через ~1 минуту сайт будет доступен по адресу:
   `https://<ваш-логин>.github.io/<название-репозитория>/`

## Технологии

- Чистый HTML + CSS + JavaScript (без фреймворков)
- [Chart.js 4.4.1](https://www.chartjs.org/) — графики
- Адаптивная вёрстка, поддержка мобильных устройств
- Автоматическое переключение светлой / тёмной темы через `prefers-color-scheme`
